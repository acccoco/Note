#### 求n次幂

可以将n写成二进制形式，所有的1位乘起来即可。这里使用了lowBit。

```cpp
// 只考虑 n>0 的情形
double myPow(double x, int n) {
    double ans = 1.0;
    int lowBit = n & (-n);
    int e = 1;
    double temp = x;		// temp恒等于x的e次方。
    while (lowBit != 0) {
        while (e < lowBit) {
            temp *= temp;
            e <<= 1;
        }
        ans *= temp;
        n &= (~lowBit);
        lowBit = n & (-n);
    }
    return ans;
}
```
