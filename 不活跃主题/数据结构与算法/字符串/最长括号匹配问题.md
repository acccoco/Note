## <b style="color:red;">最长有效括号</b>

#### <span style="color:#0202c0;">问题描述</span>

```
给定一个只包含 '(' 和 ')' 的字符串，找出最长的包含有效括号的子串的长度。
输入: ")()())"
输出: 4
解释: 最长有效括号子串为 "()()"
```

#### <span style="color:#0202c0;">解题</span>

两种情况：`...()`与`((...))`

使用`dp[i]`表示以`s[i]`结尾的最长有效括号串的长度

```cpp
// ...()
dp[i] = dp[i - 2] + 2;
// ...((...))
dp[i] = dp[i - 1] + 2 + dp[i - dp[i - 1] - 2];
```
