
#### 对象的 `id` 
每个对象都有一个 `id` 。在 `cPython` 里面， `id` 就是内存地址。
运算符 `==` 判断两个对象值是否相等；运算符 `is` 判断两个对象 `id` 是否相等


#### 不变对象
`int, string, float, tuple` 都是不变对象
以 `int` 为例来说明不变对象：
```python
a = 7      # id = 8791361197920
b = 7      # id = 8791361197920
b = b + 1  # id = 8791361197952
```

#### 可变对象
`dict, list` 都是可变对象，修改字段后其 `id` 不发生改变
